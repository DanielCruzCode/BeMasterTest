## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## To run this proyect

You need an app key to request two API's

* Youtube API
* RawgIO API
